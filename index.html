about me

background

Hi, my name is Kao Thao. I'm a Web Developer based in Fresno, CA. I like to make modern, cool and creative web designs no matter what the contents are.



I specialize in html5, css3, and Javascript creating websites for the user's experience, one at a time. I will work hard together with you everyday with love and great drive to reach success, completing your projects promptly.



I am dedicated to my craft and always strive for perfection with every new challenge.

education

I am a Web Developer intensively trained at Geekwise Academy, Fresno CA in 2016. I can design Landing Pages, build 5-page Responsive Static Websites, and create bacic CRUD Apps using Javascript. I use Github heavily to communicate and collaborate. I am comfortable using Cloud9, Brackets, and GIT from the command line.



I hold a degree in Fine Arts from CSU Fresno, which contributes tremendously to this line of work.

professional goals

My aim is to use my skills to accomplish your goals. It matters to me that the website built for you contributes to the success of your business. It is not about me.



The steps it takes to acheive success mainly depend on how each aspect of my ability is applied to our collaboration and communication. For example, a 5-Page site can be completed up to 90% within one week, each page responsive, with a fully navigational menu and all links working.



The ultimate goal is your Success and Satisfaction.

projects

dustybuns

Dustybuns was a project to test my HTML and CSS skills. More than 90% of it was completed within one week, the actual deadline goal. It was responsive with all links working. Issues were created and resolved in Github by a program manager, giving the feel of a real job experience. Though we worked in groups, the project was completed individually. One important lesson was knowing how to create dropdown menus with the input tag and positioning elements so they can be displayed correctly. Please click on the picture above and inspect its code.

geekwise academy

Geekwise Academy was the first actual project to train and strengthen my ability as a web developer. The exercise pushed me to firmly establish my HTML and CSS skills. The most important skill learned from this project was using Media Query to make the site responsive for desktop, tablet and mobile screens. Learning how to build a static website with HTML tags and manipulating them visually with CSS was challenging at first but successful and you can see when you check out the code by clicking on the picture above and inspecting it.

addressbook

Addressbook was a CRUD App created by pure vanilla Javascript. Check out its code in Github (https://github.com/geekwise-kao-thao/addressbook). The purpose was not to depend on frameworks, but to understand the fundamentals of coding so I can grasp and understand other code languages such as J-Query. When testing this app, you will find it not 100%, but its main features should demonstrate my ability to build a CRUD App simple enough for the user's experience, but complicated enough for experienced coders to appreciate.
